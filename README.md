
#### ###### # > Conversión de dinero: **
El presente programa tiene como objetivo realizar la conversión de dinero de manera precisa y eficiente a partir de una cantidad ingresada por el usuario. Por ejemplo, se puede introducir un valor numérico como 200 o 300 pesos, el cual se convertirá a la moneda seleccionada, ya sea peso mexicano o dólar estadounidense.

La interfaz principal permite al usuario seleccionar la moneda de origen y la moneda de destino mediante un menú de opciones. Una vez seleccionadas las monedas correspondientes, al ejecutar la conversión, el resultado se mostrará en la parte inferior de la interfaz.
